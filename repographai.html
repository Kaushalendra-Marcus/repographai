Features

Everything you need to
navigate unfamiliar code

Dependency Graph
An interactive force-directed graph showing every file and how it connects through imports. Click any node for details. Double-click to open the file.
AI File Summaries
Every key file gets a plain-English explanation of what it does, what it exports, and what it depends on. No more reading random files to find where things live.
Codebase Q&A
Ask anything in plain English. "Where is auth handled?", "Which files talk to the database?" The AI has full repo context and answers with specific file references.
Multi-Provider AI
Works with Groq, Ollama, Gemini, Anthropic, and OpenAI. Use any model. Type a custom model name. Switch providers anytime from the Settings tab.
Privacy First
API keys live in VS Code's encrypted SecretStorage. Your code is only sent to the provider you choose, for analysis only. No backend. No telemetry. No accounts.
Analysis History
Every analysis is saved locally per workspace. Switch between past analyses, see how the graph looked before a refactor, and delete old records when done.
How it works

Four steps from
zero to understanding

The entire pipeline runs locally. The only external call is to the AI provider you configure.

01
Scan the workspace
Reads all source files. Automatically skips node_modules, dist, build outputs, lockfiles, minified files, and generated code across all languages.
02
Build the graph
Extracts every import statement per language, resolves relative and alias paths, and constructs a node-edge dependency map of the full codebase.
03
Generate summaries
Sends batches of file content to your chosen AI provider. Returns a repo overview, architecture description, tech stack, and per-file explanations.
04
Ask anything
The Q&A agent has full context of the graph, summaries, and file contents. Multi-turn conversation with history so questions build on each other.
WorkspaceScanner — reads source files

Privacy

Your code never
leaves your machine

RepoGraph AI has no backend. There is no server, no account, no database, and no usage tracking. The only network request made is to the AI provider you explicitly configure.

Encrypted key storage
API keys are stored in VS Code's SecretStorage, which uses the OS keychain. Never written to disk in plaintext.
No telemetry
Zero usage tracking. No analytics events. No crash reports sent anywhere. The extension is fully offline-capable.
Open source and auditable
Every line of code is public. Read exactly what the extension does, verify its behavior, and contribute improvements.

Installation

Up and running in
three steps

STEP 01
Install the extension
Search for RepoGraph AI in the VS Code Extensions panel and click Install. Or install from a .vsix file directly.
ext install repograph.repograph-ai
STEP 02
Add your API key
Click the RepoGraph icon in the activity bar, open Settings, choose a provider, and paste your key. Start free with Groq.
Get a free Groq key ↗
STEP 03
Analyze your workspace
Open a project folder, go to the Analyze tab, and click Analyze Workspace. The graph and summaries will be ready in one to two minutes.
